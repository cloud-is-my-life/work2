
# [예시 과제 1] 기본 CSV 데이터 분석 (난이도: ★☆☆)

## 시나리오

운영팀은 S3에 업로드된 CSV 매출 데이터를 즉시 조회하고 싶다.
Athena로 외부 테이블을 만들고 상위 매출 데이터를 조회하시오.

---

## 요구사항

### [A] S3
- 버킷: `wsi-qfs-data`
- 경로: `sales/csv/`
- 헤더 포함 CSV

### [B] Athena
- Database: `wsi_sales_db`
- Table: `sales_csv`
- 헤더 1줄 무시

### [C] 분석
- `amount > 100000` 인 데이터 조회
- 매출 상위 10건 출력

---

## 채점 기준

| 항목 | 배점 |
|------|------|
| Database 생성 | 2점 |
| External Table 생성 | 4점 |
| 헤더 무시 적용 | 2점 |
| 상위 10건 조회 | 3점 |
| 결과 검증 | 2점 |
| **합계** | **13점** |
